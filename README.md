# 🎬 Sistema de Recomendación de Películas

![Python](https://img.shields.io/badge/Python-3.10-blue)
![ML](https://img.shields.io/badge/Machine%20Learning-Clustering-green)

Sistema de recomendación basado en clustering para agrupar usuarios y sugerir películas.

---

## 📌 Descripción

Este proyecto implementa un sistema de recomendación que:

- Agrupa usuarios por gustos
- Genera recomendaciones personalizadas
- Evalúa la calidad del agrupamiento

---

## 🧠 Tecnologías

- Python
- Scikit-learn
- Pandas
- NumPy

---

## ⚙️ Modelos

- K-Means
- Agglomerative Clustering
- DBSCAN

---

## 📊 Resultados

🏆 Mejor modelo: **K-Means**
